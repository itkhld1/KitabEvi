# KitabEvi (Verilerim Şekil Önümden Çekil)
Bu projede bozulmuş verileri temizleyerek sistemi düzgün verilerle tekrar kullanılabilir pazarlama demosuna hazır hale getiren bir mekanizmayı kitabevi online satış sitesidir. Gizli bir kısayol veya yönetici reset düğmesi ile bunu tetikleyebilir. YAZILIM MÜHENDİSLİĞİ
